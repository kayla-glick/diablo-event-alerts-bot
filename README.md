# [Alpha] Diablo 4 Event Alerts Discord Bot
A Discord Bot that notifies when World Bosses, Zone Events, and Helltides are up on https://diablo4.life.

The bot sets up a `#d4-event-alerts` channel and a `@D4 Event Alerts` role that will be pinged when events pop up.

You can use the `/d4-alerts` command to subscribe to or unsubscribe from alerts.
